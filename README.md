Usage
-----

1. Get in the MOC

	```php
	require('Api.php');
	```
2. Start your engine

	```php
	use MOC\IV\Api;
	```
3. Put the pedal to the metal.

	```php
	print Api::useModule()->useEncoding()->useText('Latin-Chars öäüß')->getTextConvertedToUtf8();
	```
